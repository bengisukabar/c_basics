# C Basics Projects

This repository contains beginner-level C programming projects created to practice fundamental programming concepts.

---

## Projects

### 1. Basic Calculator
A simple command-line calculator that performs basic arithmetic operations.

### 2. Guess the Number
A number guessing game where the user tries to guess a randomly generated number.

### 3. Number Reverse
Reverses a number entered by the user using character arrays.

---

## Purpose
- Practice C programming fundamentals
- Improve problem-solving skills
- Track learning progress through small projects
